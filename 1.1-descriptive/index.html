データの整理と要約

データを数値でとらえる基本手法を学びます。

このモジュールで学ぶこと データ分析の第一歩は、手元にあるデータを「整理して要約する」ことです。平均・中央値・分散といった基本統計量の意味と計算方法を身につけましょう。 平均・中央値・最頻値 データの「代表値」には主に3種類あります。 平均（mean）: すべての値を足して個数で割る。外れ値に敏感。 中央値（median）: 大小に並べたときの真ん中の値。外れ値に強い。 最頻値（mode）: 最も多く現れる値。 n 個のデータ の平均は （シグマ）は「全部足す」という命令記号です。 は「」を短く書いたものです。 散らばりの指標 範囲（レンジ）：最大値 − 最小値（外れ値に非常に敏感） 四分位範囲（IQR）：Q3 − Q1（外れ値に頑健） 分散・標準偏差：偏差 の2乗を平均した値 で割るのは 不偏分散（）にするためです。 が標準偏差。直感的には「標本の偏差は必ず標本平均に引き寄せられ、母集団より小さめに出る。その分を で割ることで補正する」と理解できます。 四分位数と箱ひげ図 データを小さい順に並べて4等分したときの区切り値を四分位数といいます。 Q1（第1四分位数）：下位25%の境界 Q2（第2四分位数）：中央値 Q3（第3四分位数）：上位25%の境界 箱ひげ図は Q1・Q2・Q3・最小値・最大値を1つの図にまとめます。IQR の 1.5 倍を超えた点を外れ値として識別します。 分布の形状：歪度と尖度 歪度（skewness）：分布の非対称性を表す。 - 正（右に裾が長い）→ 平均 ＞ 中央値 ＞ 最頻値 - 負（左に裾が長い）→ 平均 ＜ 中央値 ＜ 最頻値 - 0 → 左右対称（正規分布など） 尖度（kurtosis）：裾の重さ（太さ）を表す。正規分布を基準に、裾が重いほど大きな値をとります。
